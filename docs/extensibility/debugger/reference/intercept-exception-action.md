# <a name="interceptexceptionaction"></a>INTERCEPT_EXCEPTION_ACTION
Określa, jakie działania należy podjąć przy przechwytywaniu wyjątków.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
enum enum_INTERCEPT_EXCEPTION_ACTION  
{  
   IEA_INTERCEPT = 0x0001  
}  
typedef DWORD INTERCEPT_EXCEPTION_ACTION;  
```  
  
```csharp  
public enum enum_INTERCEPT_EXCEPTION_ACTION  
{  
   IEA_INTERCEPT = 0x0001  
}  
```  
  
#### <a name="parameters"></a>Parametry  
 IEA_INTERCEPT  
 Umożliwia przechwytywaniu bieżącego wyjątku. Jest to jedyna wartość obsługiwana w chwili obecnej, a musi być określona.  
  
## <a name="remarks"></a>Uwagi  
 Te wartości są przekazywane do [InterceptCurrentException](../../../extensibility/debugger/reference/idebugstackframe3-interceptcurrentexception.md) metody.
