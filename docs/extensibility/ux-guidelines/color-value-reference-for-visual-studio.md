# <a name="color-value-reference-for-visual-studio"></a>Kolor wartości odwołania dla programu Visual Studio
Ten temat jest odwołanie do tworzenia nowych tokenów kolorów. Podczas tworzenia nowego interfejsu użytkownika, można użyć podobne kolory z istniejącego interfejsu użytkownika, aby wybrać kolorów motywu każdego głównego i nazwy kolorów systemu dla tego elementu motywów duży kontrast. 

Implementacja informacji o tworzeniu nowe tokeny kolorów, zobacz [kolor narzędzia do tworzenia motywów](../../extensibility/internals/color-theming-tools.md).

Kolor tokenów do użycia z istniejącego interfejsu użytkownika, zobacz [udostępnionych kolorów dla programu Visual Studio](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

Korzystając z tego podręcznika, pamiętaj:

-   **Zawsze Kopiuj razem barwy tła i pierwszego planu.** Nie należy wybierać planu z jednego elementu interfejsu użytkownika i tła z innego wyłącznie z powodu chcesz odcienie.  

-   **Są to kody szesnastkowych, nie kolor tokenów.** Nie można użyć nazwy tokenu rzeczywiste kolorów te elementy interfejsu użytkownika, ponieważ nie można kontrolować sposób tokenów można zmienić w przyszłości. Zamiast tego utwórz tokenów w ramach własnego kategorii i użyj wartości hue wyświetlane w tabelach. Kolor tokeny dla istniejącego interfejsu użytkownika można znaleźć w [udostępnionych kolorów dla programu Visual Studio](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

- **Wysoki kontrast próbki są wyświetlane tylko porównania.** Użyj nazwy kolorów systemu, wysoki kontrast interfejsu użytkownika, a nie używaj selektora Kroplomierz lub kolorów te próbki. Wysoki kontrast próbki w tym temacie odzwierciedlają kompozycję Duży kontrast nr 1 w systemie Windows. Kolory będą wyglądać inaczej w innych motywów dużego kontrastu. 

-   **Aby uzyskać najlepsze wyniki kolor pary ustawia przy użyciu interfejsu użytkownika podobne jak w przykładzie.**
