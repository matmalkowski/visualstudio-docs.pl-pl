# <a name="browse-and-select-a-net-type-dialog-box-legacy"></a>Wyszukiwanie i wybieranie typu .NET, okno dialogowe (starsza wersja)
W tym temacie opisano sposób użycia **Wyszukaj i wybierz typ architektury .NET** okno dialogowe w starszej wersji [!INCLUDE[wfd1](../includes/wfd1-md.md)]. Użyj starszego [!INCLUDE[wfd2](../includes/wfd2-md.md)] konieczność docelowy: [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] lub [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)].  
  
 W **właściwości** okna, po wybraniu właściwości, które przyjmują typ .NET Framework w przywoływanym zestawie wielokropek **[...]**  pojawia się na końcu właściwości pola tekstowego. Klikając **[...]**  otwiera **Wyszukaj i wybierz typ architektury .NET** okno dialogowe. W tym oknie można wybrać typ z widoku drzewa w przywoływanych zestawach. Na przykład, w którym używasz Projektant działań w **właściwości** okna, kliknij przycisk **klasa bazowa** wielokropek **[...]**  aby wybrać inną klasę bazową dla działania z drzewa przywoływane zestawy.  
  
 W poniższej tabeli opisano elementy interfejsu użytkownika **Wyszukaj i wybierz typ architektury .NET** okno dialogowe.  
  
|Element interfejsu użytkownika|Opis|  
|----------------|-----------------|  
|**Nazwa typu:**|Nazwa obecnie wybranego typu.|  
|**Typ**|W okienku po lewej stronie jest wyświetlany widok drzewa zestawów odwołania. Prawe okienko Wyświetla typy, które są dostępne do wyboru z zestawu do których odwołuje się zaznaczony w okienku po lewej stronie.|
