# <a name="funcdebugstart"></a>FuncDebugStart
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [funcdebugstart —](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/funcdebugstart).  
  
Jeśli funkcja ma zdefiniowany punkcie debugowania, które jest aby rozpocząć, że punkt jest identyfikowany przez symbol `SymTagFuncDebugStart` tagu.

 W poniższej tabeli przedstawiono właściwości, które są prawidłowe dla tego typu symbolu.
