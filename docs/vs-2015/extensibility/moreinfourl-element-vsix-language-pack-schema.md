# <a name="moreinfourl-element-vsix-language-pack-schema"></a>Element MoreInfoURL (schemat VSIX Language Pack)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Najnowszą wersję tego tematu znajduje się w temacie [MoreInfoURL — Element (VSIX Language Pack schemat)](https://docs.microsoft.com/visualstudio/extensibility/moreinfourl-element-vsix-language-pack-schema).  
  
Opcjonalna. Łącze do zlokalizowanych informacji o rozszerzeniu.
