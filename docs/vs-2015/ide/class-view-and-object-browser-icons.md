## <a name="signal-icons"></a>Ikony sygnałowe  
 Następujące ikony sygnałowe dotyczą wszystkich poprzednich ikon i wskazują ich dostępność.  
  
> [!NOTE]
>  Jeśli projekt jest uwzględniona w bazie danych kontroli źródła, ikony sygnałowe dodatkowe może być wyświetlany wskazują stan kontroli źródła, takich jak zaewidencjonowany lub wyewidencjonowany.  
  
|Ikona|Opis|  
|----------|-----------------|  
|\<Brak ikony sygnału >|Public. Dostępne z dowolnego miejsca, w tym składniku i z dowolnego składnika, który odwołuje się do niej.|  
|![Sygnał chronione Symbol](../ide/media/vxsignal-icon-key.gif "vxSignal_Icon_Key")|Chronione. Dostępne z zawierającego klasy lub typu lub te pochodzące z klasy lub typu zawierającego.|  
|![Sygnał prywatnych symboli](../ide/media/vxsignal-icon-lock.gif "vxSignal_Icon_Lock")|Prywatny. Dostępne tylko w zawierającej klasy lub typu.|  
|![Sygnał zapieczętowany Symbol](../ide/media/vxsignal-icon-envelope.gif "vxSignal_Icon_Envelope")|Zapieczętowany.|  
|![Sygnał Friend&#47;wewnętrznego Symbol](../ide/media/vxsignal-icon-diamond.gif "vxSignal_Icon_Diamond")|Zaprzyjaźnione/błąd wewnętrzny. Dostępne tylko z projektu.|  
|![Sygnał ikonę strzałki](../ide/media/vxsignal-icon-arrow.gif "vxSignal_Icon_Arrow")|Skrót. Skrót do obiektu.|
