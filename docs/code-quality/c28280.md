# <a name="c28280"></a>C28280
Ostrzeżenie C28280: dla symbolu "end" znaleziono bez odpowiadającego mu "begin"

 Język adnotacji obsługuje rozpoczęcia i zakończenia (`{` i `}` w języku C) konstrukcję i kojarzenie uzyskał równowagi. Jeśli są używane makra można uniknąć tej sytuacji.
