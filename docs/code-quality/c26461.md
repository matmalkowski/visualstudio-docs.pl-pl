# <a name="c26461-useconstpointerarguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS:
  Argument wskaźnika "% argumentu %" dla funkcji "% funkcji %" może być oznaczony jako wskaźnik do `const`. Zobacz [C++ podstawowe wskazówki dotyczące con.3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
