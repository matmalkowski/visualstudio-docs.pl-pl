# <a name="c28212"></a>C28212
Ostrzeżenie C28212: Oczekiwano wyrażenia wskaźnikowego dla adnotacji  
  
 To ostrzeżenie oznacza, że parametr numerowane do adnotacji (nie funkcja adnotacją) powinien być wskaźnikiem lub tablicy typu, ale napotkano innego typu. Adnotacja musi zostać usunięty.
