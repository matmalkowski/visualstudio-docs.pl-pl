# <a name="ca0503"></a>CA0503
Właściwość jest przestarzała. Użyj właściwości zastępującej  
  
 Ostrzeżenie CA0503 jest generowany, gdy właściwość analizy kodu przestarzałe zawiera wartość w pliku konfiguracji kompilacji lub projekt. Aby usunąć to ostrzeżenie, użyj określonej właściwości, która zastępuje go.
