# <a name="mathceil-function-javascript"></a>Math.ceil — Funkcja (JavaScript)
Zwraca najmniejsza liczba całkowita większa lub równa jej argument numeryczny.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>Uwagi  
 Wymagane `number` argument jest wyrażenia liczbowego.  
  
 Wartość zwracana jest wartość całkowita równa najmniejsza liczba całkowita większa lub równa jej argument numeryczny.
